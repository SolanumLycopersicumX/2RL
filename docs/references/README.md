# References and Attribution

中文版本：[README_zh.md](README_zh.md)

This directory keeps external research material separate from authored code.

## Files

- `paper_index.md`: all papers mentioned or recommended in the research plan.
- `external_repos.md`: external codebases, install role, and where to clone.
- `reading_queue.md`: recommended reading order and note status.
- `papers/`: downloaded arXiv PDFs and `papers_manifest.csv`.

## Rules

1. Record title, link, role, and why the paper matters.
2. Keep downloaded PDFs out of git unless they are explicitly allowed and small.
3. Keep external code in `external/` or as git submodules, not copied into
   the main package.
4. Record licenses for third-party code before adapting it.
