## 一句话结论

这篇论文关注感知受损时的局部导航鲁棒性。它对 2RL 的 safety-aware routing 很重要，因为真实系统的危险往往来自感知失效，而不只是动力学扰动。

## 研究问题

深度相机、height map 或局部地图在真实环境中会有空洞、延迟、噪声和遮挡，端到端 policy 可能过度依赖错误感知。

## 核心方法

在训练中显式加入 compromised perception 或 sensor degradation，使策略学会在感知不完美时仍能安全通过。

## 环境 / 机器人 / 仿真器

腿式机器人；局部导航；感知失败场景。

## 对 2RL 的可复用点

2RL 的 OOD evaluation 应加入 sensor noise、missing height samples、stale terrain embedding。safety supervisor 可以在感知置信度低时降低速度或切 recovery。

## 注意事项

只在 clean depth/height map 上评估 MoE gate 会高估部署可靠性。
